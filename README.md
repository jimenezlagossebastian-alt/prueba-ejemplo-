🎓 Student Management System (CRUD)
Este es un sistema de gestión de estudiantes desarrollado en Python, diseñado bajo un enfoque modular. Permite realizar las operaciones básicas de un CRUD (Crear, Leer, Actualizar y Eliminar) a través de una interfaz de consola.

---

📂 Estructura del Proyecto
El proyecto se divide en dos módulos principales para mantener el código limpio y organizado:

main.py: Contiene la interfaz de usuario (menú) y el flujo principal del programa.
student_logic.py: Contiene toda la lógica de negocio, manejo de datos y funciones de procesamiento.

---

🚀 Funcionalidades
El sistema permite realizar las siguientes acciones:

Registrar Estudiantes: Captura ID, nombre, edad y curso.
Visualizar Listado: Muestra todos los estudiantes registrados actualmente.
Buscar por ID: Localiza un estudiante específico mediante su identificador único.
Actualizar Información: Permite modificar los datos de un estudiante existente.
Eliminar Registro: Borra a un estudiante del sistema por su ID.
Salir: Cierra la aplicación de forma segura.
